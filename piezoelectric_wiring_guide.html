Piezoelectric Wiring Guide — Bender Element System

Safety information

  • Never connect +20 V directly to an Arduino pin — this will permanently damage the microcontroller.
  • Verify transistor orientation and pinout against the component datasheet before powering.
  • Confirm each piezoelectric element is rated for the applied voltage.
  • Use a current-limited bench supply during initial testing.
  • Disconnect all power before modifying wiring.
  • Inspect for short circuits before energising the system.
  • Confirm oscilloscope probe ratings match the expected signal voltages.
  • Measure supply voltages with a multimeter before connecting any sensitive electronics.

Circuit schematic

Measurement concept

The two piezoelectric elements are not electrically connected to each other. The Tx piezo is part of the driver circuit — its top terminal is driven by the transistor collector node (switched +20 V) and its bottom terminal is tied to its own GND. The Rx piezo is a passive sensor — its top terminal connects only to the oscilloscope CH2 probe, and its bottom terminal is tied to its own GND.

Both elements are physically inserted into opposite ends of the soil specimen. Mechanical energy generated at Tx propagates through the soil and deflects the Rx bender element, inducing a measurable voltage. The time delay between the CH1 and CH2 waveforms is used to calculate wave velocity.

Arduino code

Upload the sketch below. It uses the built-in tone() function to generate a continuous 1170 Hz square wave on pin 8 with no additional configuration required.

Arduino (.ino) 
const int wavePin = 8; // Defined generic wave pin

void setup() {
  // Generate an 1170 Hz wave on digital pin 8
  tone(wavePin, 1170);
}

void loop() {
  // The wave runs continuously
}

Pre-power checklist

  • Confirm +20 V is isolated from all Arduino pins.
  • Confirm Arduino GND pin is connected to circuit common ground.
  • Confirm PWM signal wire runs from Arduino pin 8 to the 1 kΩ base resistor.
  • Verify Tx piezo top terminal connects to the transistor collector node; bottom terminal to its own GND.
  • Verify Rx piezo top terminal connects only to oscilloscope CH2 probe; bottom terminal to its own GND.
  • Confirm Tx and Rx piezos share no direct electrical connection.
  • Verify transistor pinout and orientation against datasheet.
  • Check piezoelectric element polarity and voltage rating.
  • Confirm oscilloscope probe connections and ratings.
  • Measure supply voltages with a multimeter before connecting electronics.
  • Inspect all wiring for shorts.
  • Set bench supply current limit before powering up.
